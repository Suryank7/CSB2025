✨ Features
---------
The Amazon Clone project includes the following features:

1. **User Authentication**: Secure user authentication and authorization using JSON Web Tokens (JWT) and React Context API.
2. **Product Browsing**: Comprehensive product browsing and searching functionality, including filtering, sorting, and pagination.
3. **Shopping Cart Management**: Robust shopping cart management and checkout functionality, including order summary and payment processing.
4. **Payment Processing**: Secure payment processing using Stripe and React Stripe.js.
5. **User Profile and Account Management**: User profile and account management, including order history and account settings.
6. **Product Reviews and Ratings**: Product reviews and ratings, including average rating calculation and review listing.
7. **Search and Filtering**: Comprehensive search and filtering functionality, including keyword searching and category filtering.
8. **Responsive Design**: Fully responsive design, ensuring a seamless user experience across a range of devices and screen sizes.

📁 Project Structure
-------------------
The project's structure is organized into the following folders:

* **public**: Publicly accessible files, including index.html and favicon.ico.
* **src**: Source code files, including JavaScript, CSS, and image assets.
* **src/components**: Reusable React components, including layout, navigation, and product components.
* **src контейнеры**: Container components, including user authentication and shopping cart management.
* **src/utils**: Utility functions, including API requests and data formatting.
* **src/styles**: CSS styles, including global styles and component-specific styles.

⚙️ How to Run
-------------
To run the application, follow these steps:

1. **Clone the repository**: Clone the repository using Git, or download the zip file.
2. **Install dependencies**: Install the required dependencies using npm or yarn.
3. **Start the application**: Start the application using npm start or yarn start.
4. **Access the application**: Access the application in your web browser at http://localhost:3000.

**Environment Variables**

* **NODE_ENV**: Set to "development" or "production" to configure the application's environment.
* **MONGO_URI**: Set to the MongoDB connection string to configure the database connection.

**Build and Deploy**

* **Build**: Run npm run build or yarn build to build the application for production.
* **Deploy**: Deploy the application to a hosting platform, such as Vercel or Netlify.

🧪 Testing Instructions
----------------------
To run the tests, follow these steps:

1. **Install dependencies**: Install the required dependencies using npm or yarn.

📦 API Reference
----------------
The application's API is documented using Swagger, and can be accessed at http://localhost:3000/api/docs.

**API Endpoints**

* **/api/products**: Retrieves a list of products.
* **/api/products/:id**: Retrieves a single product by ID.
* **/api/orders**: Retrieves a list of orders.
* **/api/orders/:id**: Retrieves a single order by ID.
